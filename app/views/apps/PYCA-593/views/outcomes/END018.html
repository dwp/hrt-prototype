Clear the PYCA to-do

  • For the question "Is a HRT interview required", select NO.
  • Click done to complete the to-do.

Clear the Refer to Decision Maker to-do

For information You'll need the biometric residence permit card for this.
  • Download the ALP.
  • Fill in section 2.
  • Upload the ALP.
  • For the question "Is this a Fast-Track HRT referral case", select YES.
  • Click done to complete the to-do.

Clear the Make a decision (HRT Fast-Track) to-do

  • For the question "Is claimant entitled?", select YES.
  • In the decision notes type "Refugee with a valid Biometric Residence Permit card".
  • Click done to complete the to-do.

Create a review and refer to decision maker (HRT)

  • Create an agent to-do.
  • Select 'Review and refer to Decision Maker (HRT)' template.
  • Input the date due (the date due is the valid until date written on the Biometric Residence Permit).
  • Select create to-do.
  • In the agent notes type 'BRP expired, review HRT required'.
  • Save notes.
  • Do NOT close this to-do.

Add a note to the claimant journal

  • Type 'HRT passed' in the claimant journal.
